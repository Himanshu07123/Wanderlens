# WanderLens

A smart web application that intelligently suggests travel destinations based on user preferences and geolocation. Discover hidden gems and popular tourist attractions with personalized recommendations.

## 🎯 Features

- **Smart Destination Suggestions**: AI-powered recommendations based on user preferences
- **User Location Detection**: Geolocation API for automatic location detection
- **Interactive Map Integration**: Leaflet/Google Maps for visual exploration
- **Nearby Tourist Place Suggestions**: Discover attractions near your location
- **Search Functionality**: Custom location search and exploration
- **Category Filters**: Browse by category:
  - 🏞️ Nature
  - 🏛️ Historical
  - 🙏 Religious
  - ⛰️ Adventure
- **Favorite Places**: Save and manage your favorite destinations (Local Storage)
- **Dark Mode**: Toggle between light and dark themes
- **Responsive Design**: Fully mobile-friendly interface
- **Modern UI**: Beautiful animations and intuitive user experience

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **APIs**: Geolocation API, Leaflet/Google Maps API
- **Storage**: Local Storage for user preferences
- **Styling**: Modern CSS with animations

## 📋 Prerequisites

- Modern web browser with JavaScript enabled
- API keys (if using Google Maps)
- Internet connection for real-time data

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Himanshu07123/Wanderlens.git
   cd Wanderlens
   ```

2. Open `index.html` in your web browser or serve with a local server:
   ```bash
   python -m http.server 8000
   ```

3. Navigate to `http://localhost:8000` in your browser

## 🎮 How to Use

1. Allow geolocation access when prompted
2. Browse suggested destinations
3. Use filters to explore specific categories
4. Click on locations to see details on the map
5. Save your favorite places
6. Toggle dark mode for comfortable viewing

## 🔧 Configuration

- Update API keys in the configuration file if using Google Maps
- Customize recommendation algorithm in `js/recommendations.js`
- Modify categories and filters in `js/config.js`
